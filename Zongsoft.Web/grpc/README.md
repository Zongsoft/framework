## Overview

[**Z**ongsoft.**W**eb.**G**rpc](https://github.com/Zongsoft/framework/tree/main/Zongsoft.Web/grpc) is the [_**g**RPC_](https://grpc.io) plugin library in the [_**Z**ongsoft_](https://github.com/Zongsoft/framework) open-source framework. It initializes the [_**g**RPC_](https://grpc.io) runtime environment for host programs.
