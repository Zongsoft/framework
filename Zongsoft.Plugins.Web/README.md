## Overview

[**Z**ongsoft.**P**lugins.**W**eb](https://github.com/Zongsoft/framework/tree/main/Zongsoft.Plugins.Web) is the _**W**eb_ plugin library in the [_**Z**ongsoft_](https://github.com/Zongsoft/framework) open-source framework. It provides plugin-based support for [_ASP.NET_](https://learn.microsoft.com/aspnet) application development.
