## Overview

[**Z**ongsoft.**P**lugins](https://github.com/Zongsoft/framework/tree/main/Zongsoft.Plugins) is a non-invasive plugin development framework. Terminal programs, rich clients, background services, and **W**eb applications can all be turned into plugin-based applications without referencing _**Z**ongsoft.**P**lugins_.


**_Documentation is being written. Follow our WeChat official account and Knowledge Planet for the latest progress and announcements._**


## Other

This framework was created in April 2010. Its implementation has gone through many iterations and refactorings, but the overall design philosophy has remained largely unchanged.
