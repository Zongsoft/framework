## 概述

[**Z**ongsoft.**D**ata.**M**ySql](https://github.com/Zongsoft/framework/tree/main/Zongsoft.Data/drivers/mysql) 是 [_**Z**ongsoft_](https://github.com/Zongsoft/framework) 开源框架中的数据引擎的底层驱动，提供了 [_**M**y**SQL**_](https://www.mysql.com) 数据库访问的相关功能，对上层应用透明，只需将该插件库部署到应用程序的插件目录中即可。
